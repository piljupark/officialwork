타이어


타이어 공기압의 점검과 조정

타이어가 냉간(최소 1시간 이상 주차 혹은 1.6km 거리 이하로 주행한 상태)

1. 타이어 밸브에서 뚜껑 분리 후 타이어 공기압 측정기 장착.(공기가 새지 않아야 함) 디지털 측정기는 공기압을 스크린에 표시. 다른 측정기에는 대부분 돌출되는 막대가 있으며, 막대에 표시된 가장 높은 압력이 현대 타이어 공기압임.

2. 정확한 수치를 위해 두 차례 반복.

3. 공기압이 낮으면 공기 펌프(주유소나 정비소에 마련되어 있음)로 호스 끝 부분을 타이어 밸브에 연결 후 공기 보충.

4. 공기압 측정기로 다시 공기압 측정. 공기압이 높을 시 타이어 밸브 스템 안쪽의 밸브를 눌러 공기 배출.

5. 타이어 공기압이 적정수준인 것이 확인되면 밸브 캡 닫기.

6. 예비 타이어를 포함해 남은 타이어에도 위의 2단계 6단계까지의 과정을 반복.

오일 교환 때마다 타이어 위치 바꾸기

타이어가 고르게 마모되도록 주기적으로 타이어 위치 바꾸기.
8,000km마다 위치 바꾸기 권장.
차체를 들어올려야 하므로 가까운 포드 서비스 센터 방문.

적절한 휠 밸런스와 휠 얼라인먼트 유지

이 작업은 타이어가 도면과 완전히 밀착될 수 있도록 해줌. 직진하는 동안 차량이 한 쪽으로 쏠리는 것이 느껴지면 얼라인먼트가 필요할 수도 있음.

타이어 마모 증상 확인하는 방법

타이어가 6년 이상 되었거나 다음과 같은 증상이 나타나면 타이어 교체 시기임.

1. 트레드 두께가 2.4mm 이하로 닳았을 때

2. 마모 표시바(타이어 트레드를 가로지르는 좁고 부드러운 고무 띠)가 나타났을 때

3. 타이어에 불룩한 곳이 생겼을 때

4. 타이어 트레드나 옆면에 구멍, 잘림, 찢어짐, 떨어짐이 있을 때

5. 트레드 홈에 균열이 생겼을 때

6. 갑자기 타이어 공기압이 낮아졌을 때

7. 미끄러운 노면에서 주행특성이나 구동력이 나빠졌을 때

8. 주행 중 갑자기 진동이 일어날 때

배터리 점검


배터리 점검이 필요한 증상

1. 배터리 배선이나 단자에 부식이나 손상이 나타날 때

2. 구동 벨트(충전기)가 낡았을 때

3. 배터리가 부풀거나 변형되었을 때

4. 배터리 배선이나 단자가 느슨해졌을 때

5. 배터리가 장착위치에 고정되지 않고 흔들리거나 움직일 때

6. 충전이나 전기장치 문제가 지속해서 일어날 때

7. 시동에 문제가 있거나 어려움을 겪을 때

올바른 배터리 선택 방법

1. 배터리 그룹 사이즈

2. 저온 시동 능력(CCA)

3. 보유용량(RC)

4. 보증내용

5. 출고 시 장착 제품 vs 애프터마켓 제품

귀하의 차량에 적합한 배터리 그룹 사이즈를 확인하려면 차량 사용설명서를 참고.

저온 시동능력(CCA)

CCA는 배터리가 섭씨 영하 18도에서 30초 동안 얼마나 충분한 전력을 공급할 수 있는지를 판단하는 수치. 대부분 차량 사용설명서에 표기되어 있거나 원래 차에 장착 되었던 배터리의 CCA 수치의 것을 권장. 고온 시동능력(HCA)이나 시동능력(CA)만 표기된 배터리를 주의할 것.

보유용량(CA)

RC는 발전기가 고장 났을 때 최저 수준 이하로 떨어지기까지 배터리가 전력을 공급할 수 있는 시간을 분 단위로 표기. 대부분 수치가 높을 수록 좋음. 시동을 끈 상태에서 차내의 전기장치를 사용한다면 이 수치를 중요하게 고려해야함.

기온 변화에 대비하기

배터리 관리에 가장 좋은 방법은 서비스 센터에서 주기적으로 점검하는 것.
배터리 관리에 대한 차량별 정보는 차량 사용설명서를 참고.

브레이크 점검


브레이크 점검이 필요한 증상

1. 삐걱거리거나 갈리는 소리가 크게 혹은 자주 들릴 때

2. 매캐하고 타는 냄새가 날 때

3. 브레이크 페달이 푹 들어가거나 발을 떼어도 복귀가 잘되지 않을 때

4. 스티어링 휠이 흔들리거나 지속해서 진동이 일어날 때

5. 브레이크에서 분진이나 금속 조각이 떨어져 나올 때

6. 브레이크 시스템 경고등이 켜질 때

서비스 센터의 브레이크 점검 과정

1. 서비스 센터에 차량 입고

2. 서비스 어드바이저가 테크니션의 진단 과정을 단계별로 안내

3. 브레이크 수리가 필요한 것으로 확인되면 곧바로 작업 진행

브레이크의 상태를 최상으로 유지하기

1. 매 예약 시 브레이크 오일 점검 및 보충

2. 매 24,000km 브레이크 종합 검사

3. 평지와 경사지에서 브레이크 페달에 발을 올린 채 운전하지 말 것

4. 연중 최소 1회 모터크래프트 브레이크 오일점검 및 보충

엔진 점검


엔진 점검이 필요한 증상

1. 시동 키를 돌렸을 때 수 초 동안 '엔진 점검 (Service Engine Soon)' 경고등이 들어오지 않을 때

2. 시동을 건 후 '엔진 점검' 경고등이 꺼지지 않을 때

3. 엔진에서 각종 오일류가 누출될 때

4. 엔진 부품에 손상이 있을 때

5. 엔진에서 큰 소음이 날 때

6. 보닛 아래에서 연기가 날 때

필터 교환


연료 필터 교체가 필요한 증상

1. 높은 엔진 회전 수에서 차가 울컥거릴 때

2. 엔진 시동이 잘 걸리지 않을 때(배터리와 점화 시스템부터 점검할 것)

연료 절감 요령


연료소비효율을 극대화하는 운전 방법

1. 계획적 주행
- 짧은 거리를 자주 움직이면 엔진이 차가운 상태로 주행. 연료소비효율을 최적화하기 위해서는 도심에서 가능한 한 여러 주행 구간을 계획적으로 잡을 것

2. 부드러운 운전
- 과격한 주행(과속, 급과속 및 급제동)은 연료 낭비의 주범. 고속 주행 때에는 최대 33%, 도심에서는 최대 5%까지 연비를 떨어뜨림. 부드러운 운전은 운전자와 다른 사람에게 안전하기도 하므로 연료비 절감 이상의 혜택을 누릴 수 있음

3. 제한 속도 지키기
- 자동차마다 효율을 내는 속도는 모두 다르지만 연료소비효율은 대부분 시속 100km 이상이 되면 빠르게 낮아짐

4. 불필요한 무게 덜어내기
- 차량에서 불필요한 것, 특히 무거운 짐을 실어두지 않아야 함. 차의 무게가 45.5kg 늘어날 때마다 연비는 최대 2%까지 낮아짐. 늘어난 무게에 연비가 저하되는 비율은 특히 큰 차보다 작은 차에서 더 큼

5. 지나친 공회전 피하기
- 대부분 차량의 배기량이 클수록 공회전 때 소비하는 연료는 작은 배기량 엔진을 얹은 차보다 많음

6. 크루즈 컨트롤 활용
- 고속도로에서 크루즈 컨트롤을 활용하면 지정된 속도가 꾸준히 유지되므로 연료 절감에 큰 기여를 함

오일 교환


오일 교환이 필요한 증상

1. 무거운 짐을 싣고 견인할 때

2. 공회전을 오래 하거나 장거리를 낮은 속도로 주행 할 때

3. 먼지가 많은 곳을 주행할 때

4. 수시로 오프로드를 주행할 때

쇼크 업소버 점검


쇼크 업소버 점검이 필요한 증상

1. 차량이 종종 튀길 때

2. 타이어가 주행 중 접지력을 잃을 때

3. 타이어 마모가 지나칠 때

4. 차량의 주행 특성이나 제동 능력이 떨어질 때

SYNC 3

1

핸즈프리 통화
버튼 클릭이나 음성 명령만으로도 차량 스피커를 통한 전화 연결을 간편하게 할 수 있습니다.

2

음악 및 엔터테인먼트
검색, 선택, 재생, 오디오북, 팟캐스트를 핸즈프리로 감상할 수 있습니다.

3

Apple CarPlay와의 연동
SYNC 3는 Apple CarPlay를 지원합니다. 시리를 이용하여 아이폰을 사용할 수 있습니다. 전화를 걸고, 음성 메시지를 확인해 보세요.

4

Android Auto와의 연동
SYNC 3는 Android Auto를 지원합니다. 구글의 음성 인식 기능인 Talk To Google을 갖추고 있으며 스마트폰과 연동이 가능합니다.

5

자동 온도조절장치
자동 온도조절장치 기능을 설치하면, 음성 인식으로 원하는 온도를 조절하거나 터치 스크린을 이용하여 손쉽게 온도를 조절할 수 있습니다.

* 일부 모델에서는 SYNC 3가 지원되지 않을 수 있습니다.

* 한국어는 지원되지 않습니다. 주행 중에는 운전에 집중하십시오.

* 일부 기능은 차량이 움직이는 중에는 사용이 제한됩니다.

ECO BOOST

1

직접 연료 분사
고압연료계통 적용으로 연료 직접 분사, 연료당 자동 조절 및 급기 온도 저감을 통해 압축비를 올렸습니다.

2

다운사이징

터보차저
